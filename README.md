# Correa

Tu asistente financiero personal con IA.

## Stack
- HTML / CSS / JS puro (sin frameworks)
- Claude API (Anthropic) para procesamiento inteligente
- Whisper API (OpenAI) para nota de voz

## Deploy
Hosteado en Vercel. Cada push a `main` hace deploy automático.

## Uso
Abrí la app, completá el onboarding de 4 pasos, y empezá a registrar gastos con voz o texto.
